# ascii-filetree

Curses file tree viewer with ASCII branches.

## Features

- Displays a directory tree
- Scroll through large trees
- Pure Python stdlib
- No dependencies

## Usage

```bash
python3 main.py
python3 main.py /path/to/folder
```

## Controls

- `j` / Down: scroll down
- `k` / Up: scroll up
- `q`: quit
